
Ce projet est une application sécurisée développée avec **Spring Boot 3.5.3**, **Spring Security**, **JWT**, et **MongoDB**. Elle expose des endpoints REST sécurisés, gère l'authentification et l'autorisation, et fournit une documentation Swagger interactive.

---

## 📌 Fonctionnalités principales

- 🔐 Authentification sécurisée avec JWT
- 🛡️ Autorisation basée sur les rôles (ex: USER, ADMIN)
- 📦 Sauvegarde des utilisateurs dans une base **MongoDB**
- 🔐 Hachage des mots de passe avec BCrypt
- 📜 Documentation Swagger disponible à `/swagger-ui.html`
- 📧 Envoi d’emails (mail service prêt à être utilisé)
- 🔍 Validation automatique des DTO (Hibernate Validator)
- 🧰 Utilisation de Lombok pour réduire le boilerplate
